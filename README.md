# cofffee_machine_project
A console-based Coffee Machine Simulator built with Python featuring resource management, payment processing, inventory tracking, and an admin panel for machine maintenance.
☕ Coffee Machine Simulator

A Python-based console application that simulates the working of a real coffee vending machine. The project allows users to order different types of coffee while automatically checking ingredient availability, processing payments, updating inventory, and managing machine resources.

✨ Features
☕ Order Espresso, Latte, and Cappuccino
🥛 Automatic resource availability checking
💰 Coffee price calculation and payment validation
🪙 Balance (change) calculation
📦 Automatic inventory updates after each order
🔒 Password-protected admin panel
🥛 Refill milk, water, coffee beans, and cups
🪙 View and update the machine's coin inventory
🖥️ Interactive menu-driven interface
